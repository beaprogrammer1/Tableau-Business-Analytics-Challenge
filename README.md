# 📊 AI Boardroom Analytics: Dashboard Enhancement Project

## 🌟 Project Overview
This project is part of a Business Analytics challenge focused on enhancing a **Sample Superstore Sales Dashboard** in Tableau. The goal is to transition from simple descriptive analytics to strategic, forward-looking insights that help C-level executives make data-driven decisions.

## 🎯 Key Objectives
- **Analyze** the existing dashboard to identify two major business insights.
- **Enhance** the workbook by adding three new strategic visualizations.
- **Propose** AI-driven predictive analytics to anticipate future outcomes.
- **Recommend** actionable solutions to maximize opportunities and minimize risks.

## 🛠️ Tools Used
- **Visualization:** Tableau Public / Tableau Desktop
- **Data Source:** Sample Superstore Dataset
- **Documentation:** Microsoft Word (for detailed report)

## 📈 New Visualizations Included
1. **Sales by Region:** A bar chart/map to identify high-performing territories.
2. **Profit Trend Over Time:** A line chart to detect periods of growth or decline.
3. **Sales vs Profit Analysis:** A scatter plot to analyze the relationship between revenue and profitability.

## 🚀 How to View
1. Download the `.twbx` file from this repository.
2. Open it in **Tableau Public** or **Tableau Desktop**.
